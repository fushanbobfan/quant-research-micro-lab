# Quant Research Micro Lab

Transparent, dependency-free building blocks for learning how backtests work.

The initial module implements a moving-average crossover strategy with a one-period execution lag to avoid look-ahead bias. It reports total return, annualized volatility, and maximum drawdown.

```powershell
python -m unittest discover -s tests
```

This project is educational software, not investment advice. Examples should use synthetic or properly licensed data.
